Detailed Description

An implementation of AuthenticationProvider that handles username/password authentication for the Codingsy Employee Management System (EMS). This provider is designed to authenticate users based on credentials provided via a UsernamePasswordAuthenticationToken.

The authentication process involves loading user details using the provided UserDetailsService and comparing the provided password with the stored encoded password using the PasswordEncoder. If the credentials are invalid, a BadCredentialsException is thrown.
