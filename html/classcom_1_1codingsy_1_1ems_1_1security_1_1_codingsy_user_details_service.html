Detailed Description

Custom implementation of Spring Security's UserDetailsService to load user-specific data. This service retrieves user details from the database and constructs a UserDetails object for authentication and authorization purposes.

It uses the UserRepository to fetch user details and maps the user's roles to Spring Security's GrantedAuthority for role-based access control.

UserDetails com.codingsy.ems.security.CodingsyUserDetailsService.loadUserByUsername ( String username) throws UsernameNotFoundException
inline

Loads the user details by username for authentication and authorization.

This method retrieves the user from the database using the provided username. If the user is not found, it throws a UsernameNotFoundException.

The user's roles are mapped to Spring Security's GrantedAuthority objects and included in the returned UserDetails object.

Parameters
usernamethe username of the user to be loaded.
Returns
a UserDetails object containing the user's details and authorities.
Exceptions
UsernameNotFoundExceptionif the user with the specified username is not found.
