\doxysubsection{Detailed Description}
Configures custom security rules for incoming requests in non-\/production environments. This class defines the security behavior for the application, including authentication mechanisms, session management, CORS, CSRF, and exception handling. It also integrates JWT-\/based authentication filters for token generation and validation.

This configuration is applied only when the "{}prod"{} profile is not active.
