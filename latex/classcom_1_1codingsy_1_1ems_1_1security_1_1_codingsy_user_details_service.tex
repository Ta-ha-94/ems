\doxysubsection{Detailed Description}
Custom implementation of Spring Security\textquotesingle{}s {\ttfamily \doxylink{}{User\+Details\+Service}} to load user-\/specific data. This service retrieves user details from the database and constructs a {\ttfamily \doxylink{}{User\+Details}} object for authentication and authorization purposes.

It uses the {\ttfamily \doxylink{interfacecom_1_1codingsy_1_1ems_1_1repository_1_1_user_repository}{User\+Repository}} to fetch user details and maps the user\textquotesingle{}s roles to Spring Security\textquotesingle{}s {\ttfamily \doxylink{}{Granted\+Authority}} for role-\/based access control.

Loads the user details by username for authentication and authorization.

This method retrieves the user from the database using the provided username. If the user is not found, it throws a {\ttfamily \doxylink{}{Username\+Not\+Found\+Exception}}.

The user\textquotesingle{}s roles are mapped to Spring Security\textquotesingle{}s {\ttfamily \doxylink{}{Granted\+Authority}} objects and included in the returned {\ttfamily \doxylink{}{User\+Details}} object.
