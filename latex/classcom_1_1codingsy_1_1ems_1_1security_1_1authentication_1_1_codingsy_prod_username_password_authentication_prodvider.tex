\doxysubsection{Detailed Description}
An implementation of {\ttfamily \doxylink{}{Authentication\+Provider}} that handles username/password authentication for the Codingsy Employee Management System (EMS). This provider is designed to authenticate users based on credentials provided via a {\ttfamily \doxylink{}{Username\+Password\+Authentication\+Token}}. 

The authentication process involves loading user details using the provided {\ttfamily \doxylink{}{User\+Details\+Service}} and comparing the provided password with the stored encoded password using the {\ttfamily \doxylink{}{Password\+Encoder}}. If the credentials are invalid, a {\ttfamily \doxylink{}{Bad\+Credentials\+Exception}} is thrown.
